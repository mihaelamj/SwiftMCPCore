# Support

Need help with swift-mcp-core? Here is where to go.

- **Questions and ideas**: open a [Discussion](https://github.com/mihaelamj/swift-mcp-core/discussions).
- **Bug reports**: open an [issue](https://github.com/mihaelamj/swift-mcp-core/issues/new/choose)
  using the bug form.
- **Feature requests**: open an issue using the feature form.
- **Security issues**: do not open a public issue. See [SECURITY.md](SECURITY.md).

Before filing, please search existing issues and discussions to avoid duplicates.

swift-mcp-core is pre-1.0 and maintained on a best-effort basis. Clear,
reproducible reports get the fastest responses.
